Import dummy data for themes developed by Kortez Themes.

== Description ==
A easy plugin to import dummy data for themes by Kortez Themes.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Kortez Toolset
3. Activate Kortez Toolset plugin from your Plugins page.
4. Go to the Appearance -> Demo Import

= From WordPress.org =

1. Download kortez-toolset.
2. Upload the 'kortez-toolset' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc.)
3. Activate Kortez Toolset from your Plugins page.

== Frequently Asked Questions ==
1. Where I can see the Settings?
Check all the available options inside Appearance > Demo Import
